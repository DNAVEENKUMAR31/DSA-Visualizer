# DSA Visualizer

**DSA Visualizer** is an interactive tool to visualize common data structures like **arrays, stacks, queues, and binary trees**. Perform operations in real-time and see how the structures change.

## Features
- Visualize **Arrays, Stacks, Queues, and Binary Trees**
- Perform standard operations (insert, delete, push, pop, enqueue, dequeue, etc.)
- **Download binary trees as SVG images**
- **Export binary tree structure as JSON**
- Interactive and beginner-friendly

## Technologies Used
- **HTML** for structure
- **CSS** for styling
- **JavaScript** for functionality and interactivity

## Usage
1. Open `index.html` in a browser.
2. Select a data structure to visualize.
3. Perform operations and see updates in real-time.
4. For binary trees, use the download buttons to export as **SVG** or **JSON**.
